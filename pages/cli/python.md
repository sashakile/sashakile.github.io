### Obtener ayuda

```sh
python --help
```

### Obtener versión

```sh
python --version
```

### Ejecutar un módulo

```sh
python -m
```

#### Crear un ambiente virtual (*virtual env*)

```sh
python -m venv directorio_ambiente_virtual
```

#### Instalar un paquete

```sh
python -m pip install nombre_del_paquete
```
